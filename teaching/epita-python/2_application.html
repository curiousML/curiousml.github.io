def fibonacci_rec(n): # recursive approach
    if n < 0:
        raise ValueError("Negative arguments not implemented")
    elif n == 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci_rec(n-1) + fibonacci_rec(n-2)

the objective is to fill a 99 grid with digits so that each column, each row, and each of the nine 33 subgrids that compose the grid (a.k.a. "boxes", "blocks", or "regions") contain all of the digits from 1 to 9. The puzzle setter provides a partially completed grid, which for a well-posed puzzle has a single solution.

step 1.¶

Define a function row_unused_digit(grid, valid_digit, i) that updates the list valid_digit so that the function checks for each column k in row i of the sudoku grid whether the digit grid[i][k] is not zero. valid_digit[k] == 0 means that k+1 is already taken in row i.

In [38]:
def row_unused_digit(grid, valid_digit, i):
    """
    Input:
    - grid:           the sudoku
    - valid_digit:    ``valid_digit[k] == 0`` means number *k+1* is already taken on this line
    - i:              row index (0 based: the index is between 0 and 8)
    """
    for k in range(9):
        if grid[i][k] > 0:
            valid_digit[grid[i][k] - 1] = 0

step 6.¶

  • If not, the Sudoku cannot be solved, go back to last list of multiple options and try the next one.
