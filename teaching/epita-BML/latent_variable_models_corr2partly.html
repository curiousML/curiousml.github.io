1. Clustering : a probabilistic point of view with k-means¶

We recall that k-means is a clustering algorithm which has the following pseudo-algorotihm :

Input : number of clusters $K$, dataset $X$

Output : the clusters

  1. Randomly initialize the parameters $\theta = \{centroid_1, \dots, centroid_K\}$

  2. Repeat until convergence :

    • For each instance $x$, compute the closest centroid : $$ k_i = \arg\min_{k} \lVert x_i - centroid_k \lVert^2 $$

    • Update the centroids : $$ centroid_k = \frac{\sum_{i:k_i=k} x_i}{|{i:k_i=k}|} $$

from sklearn.metrics import pairwise_distances
import numpy as np
# you can use "pairwise_distances" for computing the closest centroid between X and the centroids

def kmeans_naive(K, X, max_iter = 10):
    ### CODE
    
    # step 1 : let's init theta randomly
    rand_indices = np.random.choice(X.shape[0], size = K)
    centroids = X[rand_indices]
    
    # step 2 : repeat until convergence
    for _ in range(max_iter):
        cluster_distance = pairwise_distances(X, centroids, metric = 'euclidean', force_all_finite = False)
        cluster_assignment = np.argmin(cluster_distance, axis = 1)
        
        for k in range(K):
            centroids[k,:] = np.nanmean(X[cluster_assignment == k], axis = 0)
        
    return cluster_assignment

from scipy.stats import multivariate_normal
from scipy.special import softmax

def GMM_hard(K, X, max_iter = 10):
    """
    X has the standard shape : (n, d)
    """
    n, d = X.shape
    
    # Spliting the data in K sub-sets
    X_split = np.array_split(X, K)
    
    # step 1 : let's initialize pis, mus and Sigmas
    pis = np.ones(K)/K
    mus = np.array([np.nanmean(x, axis=0) for x in X_split])
    Sigmas = np.array([np.cov(x.T) for x in X_split])
    print("shape of pi", pis.shape)
    print("shape of mu", mus.shape)
    print("shape of sigma", Sigmas.shape)
    
    # step 2 : repeat until convergence
    for _ in range(max_iter):
        
        if np.sum(np.isnan(mus)):
            break
        
        # E-step : compute posteriors thx to parameters -------------------
        gamma = np.zeros((n, K)) # distribution q(T)

        for i in range(n):
            for k in range(K):
                rv = multivariate_normal(mus[k, :], Sigmas[k], allow_singular=True)
                gamma[i, k] = np.array(rv.pdf(X[i]) * pis[k])
            gamma[i, :] = softmax(gamma[i, :])
        
        # M-step : compute parameters thx to posteriors -------------------
        cluster_assignment = np.argmax(gamma, axis=1)
        
        for k in range(K):
            pis[k] = np.nanmean(gamma[:, k])
            mus[k,:] = np.nanmean(X[cluster_assignment==k], axis = 0)
            Sigmas[k] = np.cov(X[cluster_assignment==k].T)
        print(pis)
        
    return cluster_assignment

    # Spliting the data in K sub-sets
    X_split = np.array_split(X, K)

    pis = np.ones(K)/K
    mus = np.array([np.nanmean(x, axis=0) for x in X_split])
    Sigmas = np.array([np.cov(x.T) for x in X_split])
    print("shape of pi", pis.shape)
    print("shape of mu", mus.shape)
    print("shape of sigma", Sigmas.shape)
    
    # step 2 : repeat until convergence
    for _ in range(max_iter):
        
        if np.sum(np.isnan(mus)):
            break
        
        # E-step : compute posteriors thx to parameters -------------------
        gamma = np.zeros((n, K)) # distribution q(T)

        for i in range(n):
            for k in range(K):
                rv = multivariate_normal(mus[k, :], Sigmas[k], allow_singular=True)
                gamma[i, k] = np.array(rv.pdf(X[i]) * pis[k])
            gamma[i, :] = softmax(gamma[i, :])
    
        # M-step : compute parameters thx to posteriors -------------------
        for k in range(K):
            pis[k] = np.nanmean(gamma[:, k])
            mus[k,:] = np.average(X, weights=gamma[:, k], axis=0)
            Sigmas[k] = np.average(np.linalg.norm(X-mus[k], ord=2, axis=1), weights=gamma[:, k], axis=0)
    
        print(np.nanmean(gamma, axis=0))
    return np.argmax(gamma, axis=1)

2. Dimensionality reduction : a probabilistic point of view with PCA¶

PPCA is a dimensionality reduction technique that is often used when we have some missing values in the dataset. We assume that each latent variable is normally distributed : $$ t_i \sim \mathcal{N}(0, I_K) $$

The instance $x$ associated is generated by the following projection : $$ x_i | t_i \sim \mathcal{N}(W\cdot t_i + b, \sigma^2 I) $$

Question 7 : The PPCA is a general form of PCA. Indeed, we can observe it in marginalizing the latent variable. Find out the distribution of each instance $x_i | \theta$ (you can use the wikipedia table for the normal conjugacy).

reminder (useful for the E-step)

end reminder

we know that (likelihood) $x_i | t_i \sim \mathcal{N}(W\cdot t_i + b, \sigma^2I)$ and (prior) $t_i \sim \mathcal{N}(0, I)$ so by conjugate prior :

$$ p(x_i) = \int p(x_i | t_i)p(t_i)dt_i $$

is a normal distribution. Hence, we can directly compute the mean and the covariance matrix :

$$ \mathbb{E}[x_i] = \mathbb{E}[Wt_i + b + \epsilon_i] = W\mathbb{E}[t_i] + b + \mathbb{E}[\epsilon_i] = b $$$$ \begin{eqnarray} Cov(x_i) & = & \mathbb{E}[(x_i - b)(x_i - b)^T]\\ & = & \mathbb{E}[(Wt_i + \epsilon_i)(Wt_i + \epsilon_i)^T]\\ & = & W\mathbb{E}[t_it_i^T]W^T + 2W\mathbb{E}[t_i\epsilon_i^T] + \mathbb{E}[\epsilon_i\epsilon_i^T] \\ & = & WW^T+\sigma^2I = C \\ \end{eqnarray} $$

Therefore the marginal is equal to

$$ p(x_i) = \mathcal{N}(b, C) $$

Based on this marginal, it is quite easy to do the E-step !

Question 9 : Compute the M-step of the PPCA

In this question I give you directly the optimal parameters. The computation can be quite tedious (but the "idea" of the process is quite simple).

$$ \mu = \frac{1}{n}\sum_{i=1}^{n}x_i $$$$ \sigma^2 = \frac{1}{d - m}\sum_{j=m+1}^{d}\rm{eval}(S, d)_j $$$$ W = \rm{evec}(S, m)\cdot (diag(\rm{eval}(S, m)) - \sigma^2I)^{1/2} $$

with $S$ being the sample covariance, $\rm{evec}(S, m)$ the $m$ principal eigen-vectors of $S$ and $\rm{eval}(S, m)$ the $m$ principal eigen-values of $S$.
